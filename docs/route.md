## 标签管理
|  功能   | HTTP方法  | 路径 |
|  ----  | ----  | --- |
| 新增标签  | POST | /tags |
| 删除指定标签  | DELETE | /tags/:id |
| 更新指定标签  | PUT | /tags/:id |
| 获取标签列表  | GET | /tags |

## 文章管理
|  功能   | HTTP方法  | 路径 |
|  ----  | ----  | --- |
| 新增文章  | POST | /articles |
| 删除指定文章  | DELETE | /articles/:id |
| 更新指定文章  | PUT | /articles/:id |
| 获取指定  | GET | /articles/:id |
| 获取文章列表  | GET | /articles |
